In case you're getting stuck, you can (and should) use my code snapshots to compare your code to mine.

You find code snapshots for the different course sections here: https://github.com/mschwarzmueller/nextjs-course-code.

In this repository you find different "Branches" - one branch for every course section (except for the mentioned "NextJS Essentials" section).

Inside the selected branch you can browse and download the different commits to find multiple snapshots for the selected section - every commit is a snapshot that maps to some section lecture.
